## 2. Value Metric 분석

### 2.1 핵심 Value Metric 후보 평가

| Value Metric 후보 | 사용자 가치 연동성 | 측정 용이성 | 확장성 | 적합도 |
|-------------------|-------------------|------------|--------|--------|
| **AI 시스템 수** | 높음 (시스템 많을수록 컴플라이언스 부담 증가) | 높음 | 높음 | 최적 |
| 문서 생성 건수 | 중간 | 높음 | 중간 | 보조 지표 |
| 사용자 수 (시트) | 낮음 (1인 의사결정자 중심) | 높음 | 중간 | 부적합 |
| 판정 실행 횟수 | 중간 | 높음 | 낮음 | 보조 지표 |

**선정된 핵심 Value Metric: AI 시스템 수**

- 근거: AI 기본법 컴플라이언스는 "보유 AI 시스템 단위"로 의무가 발생. AI 시스템이 많을수록 영향평가서, 투명성 보고서 등 문서가 시스템별로 필요하므로, 사용자가 "더 많은 AI 시스템을 등록할수록 더 큰 가치"를 체감
- 스타트업: AI 시스템 1~3개 (Pro 충분)
- 중견기업: AI 시스템 5~20개 (Enterprise 필요)
- 대기업: AI 시스템 20개 이상 (Enterprise+ 또는 커스텀)

| 페르소나 | 현재 대안 비용 | 추정 WTP 범위 | AILEX 대비 가치 인식 | 근거 |

## 3. 추천 가격 구조

### 3.1 최종 티어 설계

### Free (무료)

**목적**: 유입 + 체험 + 바이럴 + 리드 생성

**포함 기능**:
- 고영향 AI 자가진단 1회 실행
- 판정 결과 요약 PDF 다운로드
- AI 투명성 고지 문구 생성기 (무제한)
- AI 기본법 기본 체크리스트 열람

**전략적 제한**:
- 자가진단 1회로 제한 (추가 진단 시 Pro 유도)
- 의무 문서 생성 기능 잠김 (미리보기만 가능)
- "고영향 AI 해당" 판정 시 즉시 Pro 전환 CTA 노출

**전환 유도 트리거**:
1. 판정 결과가 "고영향 AI 해당"일 때: "귀사 AI 시스템에 이행 의무가 있습니다. Pro에서 영향평가서를 5분 만에 자동 생성하세요"
2. 투명성 고지 문구 생성 후: "더 상세한 컴플라이언스 문서가 필요하다면 Pro로 업그레이드"
3. 자가진단 추가 실행 시도 시: "추가 AI 시스템 진단은 Pro에서 이용 가능합니다"

**바이럴 설계**:
- 판정 결과 공유 버튼 (LinkedIn/X/카카오톡)
- "AI 기본법 자가진단 완료" 배지 이미지 제공
- 공유 시 추가 무료 진단 1회 보너스

---

### Pro (월 30만원 / $207)

**목적**: 개인/스타트업의 핵심 컴플라이언스 니즈 완전 해결

**포함 기능**:
- 고영향 AI 자가진단 무제한
- **AI 시스템 3개** 등록/관리
- 의무 문서 AI 자동 생성 (무제한)
  - 영향평가서
  - 투명성 보고서
  - 위험관리계획서
  - 이용자 고지 문서
- 컴플라이언스 체크리스트 (시스템별)
- 컴플라이언스 대시보드
- 문서 버전 관리 + 5년 보관
- 이메일 지원

**Value Metric 한도**: AI 시스템 3개/계정

**타겟 페르소나**:
- AI 스타트업 CTO (AI 시스템 1~3개)
- 1인 AI 창업자
- 소규모 법무팀

**가격 근거**:
- 법무법인 건당 2,000만원의 **1/67**
- 과태료(최대 3,000만원) 대비 **월 1%** 비용으로 리스크 해소
- 글로벌 AI 거버넌스 도구(연 $50K+) 대비 **1/20** 가격
- 월 운영비 39만원 기준, Pro 2명이면 손익분기

---

### Enterprise (월 100만원+ / $690+ / 커스텀 견적)

**목적**: 중견기업/AI 시스템 다수 보유 기업의 종합 컴플라이언스

**포함 기능** (Pro 전체 + 추가):
- AI 시스템 **무제한** 등록/관리
- 감사 대응 모듈 (v2)
- 전문가 연결 서비스 (v2)
- 팀 기능: 멀티유저, 권한 관리 (v2)
- 우선 지원 (전용 Slack 채널)
- 커스텀 보고서
- SLA 보장

**Value Metric 한도**: 무제한

**타겟 페르소나**:
- 중견기업 법무팀 (AI 시스템 5개 이상)
- AI 시스템 다수 운영 기업

**가격 근거**:
- AI 시스템 5개 이상 시 Pro 대비 단가 효율
- 법무법인 반복 컨설팅(연 수천만원) 대비 확실한 비용 절감
- 글로벌 동급 도구(연 $50K~$500K) 대비 극적 저가

### 3.2 파운더 요금제 (Founder's Plan) -- 런칭 한정

| 항목 | 내용 |
|------|------|
| 가격 | **월 15만원** (Pro 50% 할인) |
| 조건 | 12개월 약정 (연 180만원 선결제 또는 월 결제) |
| 제한 | 선착순 **50명**, 런칭 후 60일 내 가입 |
| 추가 혜택 | "파운더 멤버" 배지, 이후 모든 신규 기능 무료, 월 1회 1:1 컴플라이언스 컨설팅 |
| 의무 | 월 1회 제품 피드백 제공 |
| 전략적 목적 | 초기 캐시플로 확보 + 피드백 루프 + 이탈 방지 (12개월 락인) |

**파운더 플랜 MRR 기여 시뮬레이션**:
- 50명 전원 확보 시: 15만원 x 50 = **월 750만원 MRR** (12개월 보장)
- 연간 확정 매출: **9,000만원**
- 이것만으로 1인 운영 비용(월 39만원) 대비 **19배 커버**

## 4. 매출 시뮬레이션

### 4.1 시나리오별 MRR 시뮬레이션 (90일 차 기준)

|----------|------------|-----------|----------|------------|-----------------|-----|
| **보수적** | 500명 | 2% | 10명 | 0명 | 20명 | 300만 + 300만 = **600만원** |
| **기본** | 1,200명 | 5% | 60명 | 2명 | 35명 | 1,800만 + 200만 + 525만 = **2,525만원** |
| **낙관적** | 2,000명 | 7% | 140명 | 5명 | 50명 | 4,200만 + 500만 + 750만 = **5,450만원** |

> 주: 파운더 플랜은 월 15만원으로 계산. Pro는 월 30만원, Enterprise는 월 100만원으로 계산.
> 실제로는 파운더 플랜 고객이 Pro 전환율에 포함되어 중복 계산 될 수 있으므로, 아래 정밀 시뮬레이션 참고.

### 4.2 정밀 시뮬레이션 (GTM 전략 기반)

GTM 전략의 퍼널 전환율 목표를 적용:

```
웹사이트 방문: 15,000명 (90일 누적)
    ↓ 가입 전환율: 8%
가입자: 1,200명
    ↓ 무료 판정 실행율: 70%
판정 실행: 840건
    ↓ "고영향 AI 해당" 비율: 50%
고영향 판정: 420건
    ↓ Pro 전환율: 17% (고영향 판정자 중)
Pro 전환: ~70명
    + Enterprise 직접 영업: 5명
    + 파운더 플랜 (Pro 할인): 35명 (70명 중 파운더 비율)
```

**90일 차 MRR 추정 (기본 시나리오)**:
- Pro 일반 (35명): 35 x 30만원 = **1,050만원**
- Pro 파운더 (35명): 35 x 15만원 = **525만원**
- Enterprise (5명): 5 x 100만원 = **500만원**
- **총 MRR: 2,075만원**

## 5. LTV/CAC 분석

### 5.1 LTV (Customer Lifetime Value) 계산

| 플랜 | 월 ARPU | 추정 평균 유지 기간 | Gross Margin | LTV |
|------|---------|-------------------|-------------|-----|
| Pro (일반) | 30만원 | 18개월 (1/churn = 1/0.055) | 80% | 30만 x 18 x 0.8 = **432만원** |
| Pro (파운더) | 15만원 | 24개월 (약정 + 높은 충성도) | 85% | 15만 x 24 x 0.85 = **306만원** |

## 8. 심리적 가격 전략

### 8.1 앵커링 (Anchoring)

**가격 테이블 배치 전략**:

```
+------------------+--------------------+---------------------+
|                  |    ★ 추천 ★        |                     |
|      Free        |       Pro          |     Enterprise      |
|     무료          |   월 300,000원     |    맞춤 견적         |
|                  |  (하루 10,000원)    |                     |
|                  |                    |   월 1,000,000원~   |
+------------------+--------------------+---------------------+
```

- **외부 앵커**: "법무법인 건당 2,000만원" -> AILEX Pro 월 30만원 (1/67)
- **내부 앵커**: Enterprise 월 100만원+을 오른쪽에 배치하여 Pro 30만원이 합리적으로 보이도록
- **시간 앵커**: "하루 10,000원" (커피 한 잔 가격) 표현
- **리스크 앵커**: "과태료 최대 3,000만원 vs 월 30만원으로 완벽 대비"

### 8.2 디코이 효과 (Decoy Effect)

현재 3-tier 구조에서 Enterprise는 디코이로 작용:
- Free: 기능 제한이 강해서 실질적 사용 어려움 -> Pro로 유도
- Enterprise: 월 100만원은 스타트업에게 과도 -> Pro가 "딱 맞는 선택"으로 인식
- **Pro가 "Goldilocks Zone"** (너무 비싸지도, 너무 적지도 않은 최적 선택)

### 8.3 프레이밍 (Framing)

| 프레이밍 기법 | 메시지 |
|-------------|--------|
| 일일 환산 | "하루 1만원으로 AI 기본법 컴플라이언스 완벽 대비" |
| 비교 프레이밍 | "법무법인 비용의 1/67" |
| 손실 회피 | "과태료 3,000만원 위험 vs 월 30만원 투자" |
| ROI 프레이밍 | "30만원 투자로 2,000만원 법무 비용 절감" |
| 시간 절약 | "2~4주 소요 수동 작업을 30분으로" |
| 사회적 증거 (출시 후) | "N개 기업이 이미 AILEX로 컴플라이언스를 해결했습니다" |

### 8.4 무료 -> 유료 전환 트리거 설계

```
사용자 여정 기반 전환 트리거:

1. [자가진단 완료] -- "고영향 AI 해당" 판정 시
   → "이행 의무가 발생했습니다. Pro에서 의무 문서를 자동 생성하세요"
   → CTA: "30일 동안 Pro를 체험하세요" (향후 무료 체험 추가 검토)

2. [투명성 고지 문구 생성 후]
   → "고지 문구만으로는 부족합니다. 영향평가서, 위험관리계획서도 필요합니다"
   → CTA: "의무 문서 전체를 한 번에 생성"

3. [판정 결과 PDF 다운로드 후]
   → "이 결과를 팀과 공유하고, 다음 단계를 시작하세요"
   → CTA: "Pro에서 컴플라이언스 대시보드 이용"

4. [2번째 AI 시스템 진단 시도 시]
   → "추가 AI 시스템 진단은 Pro 플랜에서 이용 가능합니다"
   → CTA: "AI 시스템 3개까지 관리, 월 30만원"
```

### 8.5 가격 페이지 심리 설계 원칙

1. **추천 태그**: Pro 플랜에 "가장 인기" 또는 "추천" 배지 부착
2. **대비 색상**: Pro 플랜 컬럼을 시각적으로 강조 (배경색, 테두리)

6. **고객 후기**: 가격 테이블 하단에 베타 테스터/파운더 멤버 후기 배치

## 부록: 핵심 의사결정 근거 요약

### Q: 왜 월 30만원인가?

1. **경쟁 포지셔닝**: 글로벌 AI 거버넌스 도구(연 $50K+)의 1/20 가격이면서, 법무법인(건당 2,000만원)의 1/67 비용. "저렴하되 너무 싸지 않은" 적정 가격대
2. **손익분기 현실성**: 1인 운영비 월 64만원 기준, Pro 3명이면 흑자. 30일 목표(10명)는 보수적으로도 달성 가능
3. **WTP 범위 적합**: 스타트업 CTO의 추정 WTP(월 20~50만원) 중간값
4. **심리적 경계**: 30만원은 한국 B2B에서 "팀장 결재 가능 금액" 범위 (50만원 이하는 보통 팀장 전결). 의사결정 속도가 빠름
5. **가격 인상 여지**: 30만원에서 시작하여 가치 검증 후 35~40만원으로 인상 가능 (가격 인하보다 인상이 브랜드에 유리)

### Q: 왜 Free에 자가진단 1회만 제공하는가?

1. **맛보기 효과**: 1회 체험으로 "이 도구가 동작한다"는 확신을 심어줌
2. **강제 전환 포인트**: "고영향 AI 해당" 판정 = 법적 의무 발생 = 문서 생성 필요 = Pro 필수
3. **바이럴 극대화**: 1회 진단 결과를 공유하면 추가 진단 1회 보너스 -> 공유 동기 부여
4. **무분별한 무료 사용 방지**: 무제한 무료 시 "무료로 충분"하다는 인식 형성 방지

### Q: 파운더 플랜 50% 할인이 브랜드를 훼손하지 않는가?

- 파운더 플랜은 "한정 50명, 60일 한정"이라는 희소성으로 정상 가격의 가치를 오히려 강화
- "50명 이후에는 30만원"이라는 메시지가 30만원을 앵커로 설정
- 12개월 약정으로 연간 180만원 확보 -> 실질적으로는 할인이 아닌 "장기 계약 인센티브"
