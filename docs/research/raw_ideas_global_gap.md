# AI 사업 아이디어 리서치: "해외에서 검증됐는데 한국에 없는 것"

> 리서치 날짜: 2026-02-12
> 접근 방식: 해외 $10M+ 매출 검증 모델 중 한국 시장 공백 탐색
> 6가지 조건: 해외 검증 + 한국 공백 + 예창패 딥테크(AI) + 1인 3개월 MVP + 필수 수요 + 재미

## 아이디어 1: AI 기본법 컴플라이언스 자동화 플랫폼

### 해외 검증 사례
- **Credo AI** (미국): AI 거버넌스 플랫폼, 소프트웨어 매출 ~3x 성장, 고객 2배 증가, 100% 리텐션. Forrester Wave 2025 리더 선정. Gartner Market Guide 2025 등재.
- **시장 규모**: AI 거버넌스 시장 2025년 $309M -> 2034년 $4.8B (CAGR 35.7%)
- **OneTrust**: AI Governance 모듈로 한국 시장 공략 시도 중이나 한국법 특화 부족

### 한국 경쟁 현황
- **거의 공백 상태**. Startup Alliance 조사에 따르면 한국 AI 스타트업 중 단 2%만이 AI 기본법 컴플라이언스 프레임워크를 사전 준비.
- 슬렉슨(Slaxon)이 온프레미스+추적성으로 접근하지만, 자동화된 컴플라이언스 플랫폼은 부재.
- 글로벌 솔루션(OneTrust, Credo AI)은 한국 AI 기본법 특화 기능 없음.
- **한국이 세계 최초로 AI법을 전면 시행(2026.1.22)한 국가** -> 한국 특화 솔루션 필수.

### 예창패 적합도: ★★★★★
- **딥테크 분야**: AI (빅데이터/AI 직접 해당)
- **사회문제 해결**: AI 안전성/신뢰성 확보, 국민 기본권 보호
- 세계 최초 AI법 전면 시행 국가로서의 글로벌 확장 잠재력 -> 심사위원 어필 최상급

### 1인 MVP 범위 (3개월)
1. **AI 영향평가 자동 생성기**: 기업이 자사 AI 서비스 정보를 입력하면 고영향 AI 해당 여부를 자동 판정하고, 필요한 투명성/안전성 문서를 자동 생성
2. **컴플라이언스 체크리스트 대시보드**: AI 기본법 10개 고영향 영역별 의무사항 자동 매핑
3. **AI 표시 의무 관리 도구**: 생성형 AI 산출물에 대한 표시 의무 관리

### "안 쓰면 손해"인 이유
- **법적 의무**: 2026.1.22부터 시행 중. 고영향 AI 사업자는 안전성/투명성 확보 의무 위반 시 제재.
- 기업들의 "어떤 서비스가 고영향 AI에 해당하는지" 해석 불명확 -> 자동 판정 도구 절실.
- 대기업은 법무팀이 하겠지만, **AI 스타트업/중소기업은 전담 인력 없음** -> SaaS 도구 필수.

### 수익 모델과 가격
- **프리미엄**: 무료 체크리스트 + 유료 자동 문서 생성
- 스타트업: 월 29만원 (AI 서비스 5개까지)
- SME: 월 99만원 (무제한 + 감사 보고서)
- 엔터프라이즈: 월 299만원+ (커스텀 + API + 컨설팅)
- **예상 TAM**: 한국 AI 기업 약 3,000개 x 평균 월 50만원 = 연 180억원

## 아이디어 2: AI ESG/탄소 회계 자동화 플랫폼 (중소기업 특화)

### 해외 검증 사례
- **Watershed** (미국): $100M 시리즈C, $1.8B 밸류에이션 (2024.2). 탄소 회계 + 감축 계획 자동화.
- **Persefoni** (미국): $23M 시리즈C (2025.4). AI 기반 탄소 발자국 관리 소프트웨어.
- **Sweep** (프랑스): $100M 펀딩. 유럽 CSRD 대응 특화.
- **시장 규모**: 글로벌 탄소 회계 시장 2025년 -> 수십억 달러 규모, CAGR 30%+

### 한국 경쟁 현황
- **그린플로(오후두시랩)**: ERP 연동 자동 탄소 계산. 하지만 대기업 제조업 중심.
- **하나에코(하나루프)**: ESG 플랫폼. 종합적이나 중소기업 특화 부족.
- **땡스카본**: ESG 공시 현황 분석. 컨설팅 중심.
- **핵심 공백**: 대기업 공급망 내 **중소 협력사를 위한 Scope 3 배출량 자동 보고 도구**가 없음.

### 예창패 적합도: ★★★★★
- **딥테크 분야**: AI + 빅데이터
- **기후테크 가점**: 직접 해당! 탄소 감축 지원
- **사회문제**: 기후변화 대응, ESG 의무화
- 딥테크 + 기후테크 이중 가점 -> 예창패 최적 아이템

### 1인 MVP 범위 (3개월)
1. **Scope 1,2,3 자동 계산기**: 회사 활동 데이터(전기요금, 연료비, 출장비 등) 입력 -> AI가 배출량 자동 산정
2. **공급망 데이터 수집 자동화**: 대기업 협력사에 자동 설문 발송 -> 미응답 시 AI가 산업 평균으로 추정
3. **한국 배출권거래제 보고서 자동 생성**: 한국 규정에 맞는 양식 자동 채움

### "안 쓰면 손해"인 이유
- **2026년 자산 2조원 이상 KOSPI 상장사 ESG 공시 의무화** -> 대기업은 공급망(Scope 3) 데이터를 협력사에게 요구
- 협력사 데이터 측정/취합의 어려움이 63%로 1위 애로사항
- 중소기업은 대기업 거래 유지를 위해 탄소 보고를 **반드시** 해야 함 -> 거래관계 생존 문제
- 2030년 전체 KOSPI 상장사 확대 -> 시장 지속 성장

### 수익 모델과 가격
- **타겟**: 대기업 공급망 내 중소 협력사 (수만 개)
- 소기업: 월 19만원 (Scope 1,2 자동화)
- 중기업: 월 49만원 (Scope 3 포함 + 보고서)
- 대기업 공급망 패키지: 협력사당 월 9만원 (대기업이 협력사에 제공)
- **예상 TAM**: KOSPI 대기업 협력사 약 50,000개 x 월 15만원 = 연 900억원

## 아이디어 3: AI UGC 동영상 광고 생성 플랫폼 (한국 이커머스 특화)

### 해외 검증 사례
- **Arcads** (프랑스): 런칭 수개월만에 수익성 달성, 7명 팀으로 $10M+ ARR 돌파. 6,000+ 고객, 월 100,000+ 에셋 생성. $16M 시드 라운드 (2025.12).
- **Synthesia** (영국): $100M+ ARR, $4B 밸류에이션. Fortune 100의 70%+ 사용.
- **HeyGen** (미국): $100M+ ARR (2025.10). 2023년 $1M -> 2025년 $100M으로 100배 성장.
- **시장 규모**: AI in Creator Economy 시장 2025년 $4.35B, 2029년 $12.85B (CAGR 31.1%)

### 한국 경쟁 현황
- **브이캣(VCAT)**: 상품 URL로 숏폼 영상 자동 생성. 하지만 AI 아바타/UGC 스타일은 아님.
- **뤼튼**: 마케팅 카피 생성 위주. 영상 생성 약함.
- **핵심 공백**: 한국 이커머스 셀러(스마트스토어/쿠팡) 특화 AI UGC 동영상 광고 생성 도구 없음.
  - 한국인 얼굴의 AI 아바타
  - 한국어 자연스러운 말투
  - 스마트스토어/쿠팡/인스타그램/틱톡 최적 포맷 자동 변환
  - 상품 URL만 넣으면 리뷰형 UGC 영상 자동 생성

### 예창패 적합도: ★★★★☆
- **딥테크 분야**: AI (생성형 AI, 딥러닝 아바타)
- 소상공인/1인 셀러 지원 -> 중소기업 성장 기여
- 기후테크 가점은 없으나, 기술적 깊이(립싱크, 음성 합성, 영상 생성)가 높음

### 1인 MVP 범위 (3개월)
1. **한국인 AI 아바타 UGC 영상 생성기**: 상품 URL 입력 -> AI가 리뷰 스크립트 생성 -> AI 아바타가 말하는 숏폼 영상 자동 제작
2. **플랫폼별 최적화**: 스마트스토어(9:16 세로형), 쿠팡(정사각형), 틱톡/인스타(15-60초)
3. **A/B 테스트**: 여러 버전 자동 생성 -> 어떤 스타일이 클릭률 높은지 분석

### "안 쓰면 손해"인 이유
- UGC 광고의 전환율이 일반 광고 대비 4-5배 높음 (글로벌 데이터)
- 실제 리뷰어 촬영 비용: 건당 10-50만원. AI 생성: 건당 1만원 미만 -> 비용 90%+ 절감
- 경쟁 셀러가 AI UGC를 쓰기 시작하면 안 쓰는 셀러는 노출에서 밀림
- 쿠팡/스마트스토어 알고리즘이 동영상 콘텐츠에 가산점 부여

### 수익 모델과 가격
- **타겟**: 한국 이커머스 셀러 약 50만명+ (스마트스토어 + 쿠팡)
- 스타터: 월 3.9만원 (월 10개 영상)
- 프로: 월 9.9만원 (월 50개 영상 + A/B 테스트)
- 비즈니스: 월 29.9만원 (무제한 + API + 팀 기능)
- **예상 TAM**: 활성 셀러 50만 x 5% 전환 x 월 7만원 = 연 210억원

## 아이디어 4: AI 음성 에이전트 플랫폼 (한국 소상공인 전화 응대 자동화)

### 해외 검증 사례
- **Bland AI** (미국): 인바운드 $0.04/분, 아웃바운드 $0.09/분. 엔터프라이즈 연 $150K+.
- **Vapi** (미국): $130M 밸류에이션, 시리즈A $20M (2024). YC 출신. $0.05/분 플랫폼 수수료.
- **Synthflow**: AI 음성 에이전트 SaaS. 중소기업 타겟.
- **시장 규모**: 글로벌 음성 AI 에이전트 시장 2024년 $2.4B -> 2034년 $47.5B (CAGR 34.8%)

### 한국 경쟁 현황
- **디윅스(Dewix)**: AI 에이전트 자동화. 하지만 엔터프라이즈 중심.
- **클레버247.ai**: 24/7 영업/고객지원 자동화. 글로벌 서비스로 한국어 최적화 약함.
- **핵심 공백**: **한국 소상공인(식당, 미용실, 병원, 학원) 전화 예약/문의 자동 응대** AI 에이전트가 없음.
  - 한국어 자연스러운 음성 (존댓말/반말 구분)
  - 네이버 예약/카카오톡 연동
  - 전화번호 그대로 유지하며 AI 자동 응대
  - 전화 -> 카카오톡 예약 확인 메시지 자동 발송

### 예창패 적합도: ★★★★☆
- **딥테크 분야**: AI (음성 인식, 자연어 처리, 음성 합성)
- 소상공인 인력난 해결 -> 사회문제
- 기후테크 가점은 없으나, 기술 난이도가 높음 (한국어 실시간 대화)

### 1인 MVP 범위 (3개월)
1. **전화 자동 응대 봇**: 전화가 오면 AI가 자동 응대. "예약하시겠습니까?" -> 날짜/시간/인원 확인 -> 예약 등록
2. **카카오톡 연동**: 예약 확인 메시지 카카오톡으로 자동 발송
3. **관리 대시보드**: 전화 기록, 예약 현황, 놓친 전화 알림

### "안 쓰면 손해"인 이유
- 소상공인 평균 하루 놓치는 전화: 10-30통. 각 통화 = 잠재 매출 2-5만원 -> 하루 20-150만원 손실
- 직원 한 명 고용(월 230만원+) vs AI 에이전트(월 10만원) -> 비용 95% 절감
- 바쁜 시간대(런치타임, 퇴근 후) 전화 놓치면 경쟁 업체로 고객 이탈
- 네이버 예약 시스템은 있지만 **전화 응대**는 자동화되지 않음

### 수익 모델과 가격
- **타겟**: 한국 소상공인 약 330만개 (요식업, 미용, 의료, 교육)
- 베이직: 월 4.9만원 (월 200통)
- 프로: 월 9.9만원 (월 500통 + 카카오톡 연동)
- 프리미엄: 월 19.9만원 (무제한 + 맞춤 시나리오)
- **예상 TAM**: 소상공인 330만 x 3% 전환 x 월 8만원 = 연 950억원

## 아이디어 5: AI 공급망 Scope 3 데이터 수집 에이전트 (대기업 납품용)

### 해외 검증 사례
- **Resilinc** (미국): Agentic AI 공급망 플랫폼. Gartner 2025 Magic Quadrant 리더. $250M 시리즈E.
- **Pactum AI**: 자율 AI 에이전트가 공급업체와 자동 협상.
- **시장 규모**: AI in Supply Chain 시장 CAGR 24%+

### 한국 경쟁 현황
- **그린플로**: 탄소 자동 계산이지만 공급망 데이터 수집 자동화는 약함.
- **핵심 공백**: 대기업이 수백 개 협력사로부터 ESG/탄소 데이터를 자동 수집하는 AI 에이전트가 없음.

### 예창패 적합도: ★★★★★
- **딥테크**: AI 에이전트
- **기후테크 가점**: 직접 해당 (공급망 탄소 관리)
- B2B이지만 대기업 단위로 수백 개 협력사에 배포되므로 확장성 높음

### 1인 MVP 범위 (3개월)
1. **AI 데이터 수집 에이전트**: 대기업이 시스템에 협력사 리스트 등록 -> AI가 각 협력사에 이메일/카카오톡으로 데이터 요청 -> 응답을 자동 파싱/검증
2. **미응답 자동 추정**: 응답 없는 협력사는 산업 평균 데이터로 AI 추정 + 신뢰구간 제공
3. **대시보드**: 대기업 ESG팀이 전체 공급망 탄소 현황을 한 눈에 파악

### "안 쓰면 손해"인 이유
- 2026년 ESG 공시 의무화. Scope 3가 총 배출량의 70-90% 차지.
- 설문 조사에서 "협력업체 데이터 측정/취합 어려움"이 63%로 1위 애로사항.
- 수작업으로 100개 협력사 데이터 수집 = 담당자 1명 풀타임 3개월. AI = 3일.

- 한국 중소기업 영업사원 1인 월급: 300-500만원. AI SDR: 월 30-50만원 -> 90% 절감
- 영업사원 1명이 하루 보내는 이메일: 20-30통. AI: 200-300통 -> 10배 효율
- 경쟁사가 AI 세일즈를 도입하면 같은 시장에서 밀림
- 한국 중소기업의 가장 큰 고민: "영업/마케팅 인력 부족"

### 수익 모델과 가격
- **타겟**: B2B 영업이 필요한 한국 중소기업 약 50만개
- 스타터: 월 19.9만원 (월 1,000통 이메일)
