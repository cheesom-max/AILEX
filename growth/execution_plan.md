- 담당: [이름]

"[행동]하면 [지표]가 [현재 수치]에서 [목표 수치]로 [N]% 개선될 것이다."

### 콜드 이메일 관련

| 규정 | 요구사항 | AILEX 대응 |

| GDPR (해외 발송 시) | 정당한 이익(Legitimate Interest) 근거 | B2B 업무 관련 이메일로 정당한 이익 주장 가능 |
| CAN-SPAM (미국) | 물리적 주소 포함, 10일 내 수신거부 처리 | 해당 시 준수 |

| 분석 데이터 | 익명화된 집계 데이터만 마케팅에 활용 |
