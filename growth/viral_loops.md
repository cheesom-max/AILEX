## 바이럴 루프 1: 내재적 바이럴 -- "컴플라이언스 배지"

### 개요
AILEX에서 모든 의무사항을 이행 완료한 기업에게 "AI 기본법 준수 기업" 배지를 발급합니다. 기업은 이 배지를 자사 웹사이트, 이메일 서명, 마케팅 자료에 삽입하여 신뢰도를 높일 수 있습니다. 배지를 클릭하면 AILEX 웹사이트로 연결됩니다.

### 메커니즘

```
기업이 AILEX로 컴플라이언스 완료
        ↓
"AI 기본법 준수 기업" 배지 발급
        ↓
기업이 배지를 웹사이트/이메일 서명에 삽입
        ↓
배지를 본 거래처/파트너/투자자가 "우리도 해야 하나?" 인식
        ↓
배지 클릭 -> AILEX 무료 자가진단 랜딩 페이지
        ↓
신규 사용자 가입 -> 자가진단 -> Pro 전환
        ↓
새 기업도 배지 획득 -> 루프 반복
```

### 구체적 구현 방법

1. **배지 디자인**: "Verified by AILEX | AI 기본법 준수" 로고 + 검증 날짜
2. **임베드 코드**: `<a href="https://ailex.ai/verify/{{company_id}}"><img src="배지URL" /></a>`
3. **검증 페이지**: 배지 클릭 시 해당 기업의 컴플라이언스 현황 확인 가능 (공개 범위는 기업이 설정)
4. **배지 유지 조건**: Pro 플랜 활성 구독 + 최근 6개월 내 문서 업데이트
5. **SEO 효과**: 배지에 nofollow 없는 백링크 포함 -> AILEX 도메인 권위 강화

### 예상 효과

- **노출 범위**: Pro 사용자 기업의 웹사이트 방문자에게 자연 노출
- **클릭률**: 배지 노출 대비 1~3% CTR 예상
- **전환**: 클릭 후 무료 자가진단 실행률 30~50%
- **K-factor 기여**: 0.1~0.2 (유료 사용자 1명 -> 0.1~0.2명 신규 유입)
- **추가 효과**: B2B 거래에서 "AI 컴플라이언스 인증"이 요구 조건이 되면 바이럴 효과 폭발적 증가

### B2B 거래 시나리오

```
대기업 A사 구매팀: "AI 솔루션 공급업체는 AI 기본법 준수 인증 필요"
        ↓
공급업체 B사 (아직 미대응): "인증을 받아야 계약이 가능하네?"
        ↓
B사가 AILEX 가입 -> 컴플라이언스 완료 -> 배지 획득
        ↓
B사의 다른 거래처 C사도 같은 요구를 받음 -> AILEX 가입
```

## 바이럴 루프 2: 인센티브 바이럴 -- "추천 보상 프로그램"

### 개요
기존 유료 사용자가 동료/지인에게 AILEX를 추천하면, 추천인과 피추천인 모두에게 혜택을 제공하는 양면(two-sided) 추천 프로그램입니다.

### 메커니즘

```
기존 유료 사용자가 추천 링크 생성
        ↓
추천 링크를 동료/커뮤니티에 공유
        ↓
피추천인이 링크로 가입 + 무료 자가진단 실행
        ↓
피추천인이 Pro 전환 시:
  - 추천인: 다음 달 구독료 50% 할인 (15만원)
  - 피추천인: 첫 달 30% 할인 (21만원)
        ↓
피추천인도 추천 링크 생성 -> 루프 반복
```

### 보상 구조

| 조건 | 추천인 혜택 | 피추천인 혜택 |
|------|-----------|-------------|
| 피추천인 가입 (무료) | 추가 무료 진단 1회 | 기본 무료 진단 |
| 피추천인 Pro 전환 | 다음 달 구독료 50% 할인 | 첫 달 30% 할인 |
| 누적 3명 Pro 전환 | 1개월 무료 | - |
| 누적 5명 Pro 전환 | 3개월 무료 + "AILEX Ambassador" 배지 | - |
| 누적 10명 Pro 전환 | 6개월 무료 + 연간 플랜 VIP 가격 영구 적용 | - |

### 구체적 구현 방법

1. **추천 대시보드**: 인앱에서 고유 추천 링크 생성 + 추천 현황(클릭/가입/전환) 실시간 확인
2. **추천 링크 형태**: `https://ailex.ai/?ref={{user_id}}` + UTM 파라미터
3. **쿠키/세션 트래킹**: 30일 어트리뷰션 윈도우
4. **알림**: 피추천인 가입 시/전환 시 추천인에게 이메일 + 인앱 알림
5. **소셜 공유 버튼**: LinkedIn, X, 카카오톡 원클릭 공유

### K-factor 목표 계산

```
K-factor = 초대 수 x 전환율

목표 시나리오:
- 유료 사용자 중 추천 참여율: 20%
- 참여자당 평균 초대 수: 3명
- 초대받은 사람의 가입률: 30%
- 가입자의 Pro 전환율: 15%

K = (유료 75명 x 20% 참여) x 3 x 30% x 15%
K = 15 x 3 x 0.3 x 0.15
K = 2.025명 추가 유료 전환

K-factor = 2.025 / 75 = 0.027 (보수적)

낙관적 시나리오 (참여율 30%, 초대 5명):
K = 22.5 x 5 x 0.3 x 0.15 = 5.06명
K-factor = 5.06 / 75 = 0.067
```

K-factor가 1 미만이므로 자체적으로 바이럴은 아니지만, 다른 채널과 결합하면 유의미한 추가 성장 동력이 됩니다.

### 예상 효과

- 월 추가 유료 전환: 2~5명 (Pro 기준)
- 연간 추가 MRR: 60만~150만원
- 추천을 통한 고객의 LTV가 직접 획득 고객 대비 16% 높음 (업계 벤치마크)

## 바이럴 루프 3: 콘텐츠 바이럴 -- "자가진단 결과 공유"

### 개요
무료 자가진단 결과를 소셜 미디어에 공유할 수 있는 기능을 제공합니다. "우리 회사는 AI 기본법 자가진단을 완료했습니다"라는 메시지와 함께 결과 카드 이미지가 자동 생성됩니다.

### 메커니즘

```
사용자가 무료 자가진단 완료
        ↓
결과 페이지에 "이 결과를 공유하기" 버튼 표시
  - LinkedIn / X / 카카오톡 / 링크 복사
        ↓
공유 시 자동 생성되는 콘텐츠:
  "{{회사명}}은 AI 기본법 자가진단을 완료했습니다.
   결과: [고영향 AI 해당/해당 없음]
   당신의 AI 서비스는? -> 무료 진단하기: [링크]"
        ↓
공유 글을 본 동료/팔로워가 "우리도 해봐야겠다" 인식
        ↓
링크 클릭 -> AILEX 자가진단 실행 -> 루프 반복
```

### 구체적 구현 방법

1. **결과 카드 이미지**: OG 이미지 자동 생성 (회사명 + 판정 결과 + 날짜 + AILEX 로고)
2. **공유 텍스트 자동 생성**: 플랫폼별 최적화된 문구 (LinkedIn은 전문적, X는 간결, 카카오톡은 친근)
3. **공유 인센티브**: 공유 1회당 추가 무료 진단 1회 제공
4. **바이럴 계수 트래킹**: 공유 수 / 자가진단 완료 수 = 공유율

### 플랫폼별 공유 문구

**LinkedIn**:
```
AI 기본법이 시행된 지 한 달이 지났습니다.

오늘 AILEX의 무료 자가진단을 통해 우리 서비스가
고영향 AI에 해당하는지 확인했습니다.

98%의 AI 기업이 아직 대응 체계를 갖추지 못한 상황에서,
5분 만에 현황을 파악할 수 있다는 게 인상적이었습니다.

AI 서비스를 운영하는 분들은 한번 해보시길 추천합니다.
무료 자가진단: [링크]

#AI기본법 #컴플라이언스 #고영향AI #AILEX
```

**X (Twitter)**:
```
AI 기본법 자가진단 5분 만에 완료.
우리 서비스가 고영향 AI에 해당하는지 바로 확인 가능.

98%가 아직 미대응이라는데, 무료니까 한번 해보세요.
[링크]

#AI기본법 #AILEX
```

**카카오톡**:
```
[AI 기본법 자가진단 해봤는데]

5분이면 끝나고, 우리 서비스가 고영향 AI인지 바로 알 수 있어.
무료야. 한번 해봐!
[링크]
```

### 예상 효과

- 자가진단 완료자 중 공유율: 10% (인센티브 적용 시 15%)
- 공유 1건당 평균 클릭: 5~15건
- 클릭 후 자가진단 실행율: 30~50%
- **월 90일 차 기준**: 자가진단 1,000건 x 10% 공유 = 100건 공유 x 10건 클릭 = 1,000추가 클릭 x 40% 실행 = 400건 추가 자가진단
- K-factor 기여: 0.3~0.5 (무료 사용자 기준)

## 바이럴 루프 종합 예상 효과

| 루프 유형 | K-factor | 주요 지표 | 구현 난이도 | 우선순위 |
|-----------|----------|----------|------------|----------|
| 내재적 (배지) | 0.1~0.2 | 배지 노출 -> 클릭 -> 가입 | 중간 (배지 시스템 구축) | P1 |
| 인센티브 (추천) | 0.03~0.07 | 추천 -> 가입 -> 전환 | 중간 (추천 대시보드 구축) | P1 |
| 콘텐츠 (공유) | 0.3~0.5 | 공유 -> 클릭 -> 자가진단 | 낮음 (공유 버튼 + OG 이미지) | **P0** |

**통합 K-factor 목표**: 0.43~0.77

**구현 순서**:
1. (즉시) 콘텐츠 바이럴: 자가진단 결과 공유 기능 -- 가장 쉽고 빠름
2. (런칭 후 2주) 인센티브 바이럴: 추천 보상 프로그램
3. (런칭 후 1개월) 내재적 바이럴: 컴플라이언스 배지 시스템

|------|----------|----------|----------|
