## 1. AI 투명성 고지 의무란?

### 법적 근거
- **AI 기본법 제40조**: AI 사업자는 사용자에게 AI 사용 사실을 명확히 고지해야 함
- **위반 시 과태료**: 500만원 ~ 1,500만원

### 고지 의무 3가지 원칙
1. **명확성**: 사용자가 쉽게 이해할 수 있는 언어
2. **가시성**: 사용자가 즉시 볼 수 있는 위치
3. **사전 고지**: AI 사용 전에 고지

## 3. 고지 문구 배치 위치 가이드

### 원칙: 사용자가 AI 사용 전에 즉시 볼 수 있어야 함

### 웹사이트
- **위치**: 서비스 시작 페이지 상단 또는 팝업
- **크기**: 최소 12px 이상
- **색상**: 배경과 대조되는 색상 (흑백 대비 4.5:1 이상)

### 모바일 앱
- **위치**: 앱 첫 실행 시 온보딩 화면
- **형태**: 전체 화면 또는 바텀 시트
- **확인 버튼**: "AI 사용에 동의합니다" 버튼 필수

### 물리적 환경 (키오스크, ATM)
- **위치**: 화면 상단 고정 배너
- **언어**: 한국어 + 영어 병기 권장

## 4. 생성형 AI 표시 의무 (제43조)

생성형 AI가 생성한 콘텐츠는 **반드시 표시**해야 합니다.

### 이미지
```
워터마크 예시:
"AI Generated" 또는 "AI 생성"
위치: 이미지 하단 모서리
크기: 이미지 전체 크기의 5% 이상
```

### 텍스트
```
표시 예시:
"본 글은 인공지능이 작성했습니다."
위치: 텍스트 상단 또는 하단
형식: 굵은 글씨 또는 배경색 강조
```

### 음성/영상
```
표시 예시:
"이 음성/영상은 AI가 생성했습니다."
위치: 영상 시작 전 3초 이상 표시
형식: 자막 또는 음성 고지
```

## 5. 자주 묻는 질문 (FAQ)

### Q1. 고지 문구를 이용약관에만 적으면 되나요?
**A**: 안 됩니다. 이용약관은 사용자가 읽지 않는 경우가 많으므로, **서비스 화면에 명확히 표시**해야 합니다.

### Q2. "AI 기술 활용"이라고만 써도 되나요?
**A**: 권장하지 않습니다. **"인공지능 기술을 사용합니다"**처럼 명확한 표현이 필요합니다.

### Q3. 영어로만 써도 되나요?
**A**: 한국 사용자 대상이라면 **한국어 필수**입니다. 영어 병기는 선택 사항입니다.

### Q4. AI 고지 문구가 너무 길면 사용자 경험이 나빠지지 않나요?
**A**: "자세히 보기" 버튼으로 접거나, 한 번만 표시 후 "다시 보지 않기" 옵션을 제공할 수 있습니다.

### Q5. 생성형 AI 표시를 안 하면 과태료가 얼마인가요?
**A**: 500만원 ~ 1,500만원입니다.
