## 6. SEO 최적화 전략

### 6.1 On-Page SEO
- **Title 패턴**: "[키워드] 완벽 가이드 | AILEX"
- **Meta Description 패턴**: "[키워드]를 5분 만에 해결하세요. 법무법인 2,000만원 vs AILEX 월 30만원. 지금 무료 자가진단 시작 →"
- **H1 패턴**: "키워드 + 실무 해결책"
- **내부 링크**: 가이드 → 도구 소개 → Free 플랜 → Pro 플랜

### 6.2 구글 AI Overview (GEO) 최적화
- FAQ 스키마 마크업 적용
- 답변형 콘텐츠 구조 (질문 → 핵심 답변 → 상세 설명)
- "AI 기본법이란?" → 즉시 답변 가능한 200자 요약

### 6.3 네이버 스마트블록 최적화
- 표/리스트 형식 콘텐츠
- "AI 기본법 11개 고영향 영역" → 번호 리스트
- "AI 기본법 의무사항 체크리스트" → 체크박스 형식
